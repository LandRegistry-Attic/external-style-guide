Land Registry style guide

This guide demonstrates styles and modules for Land Registry applications.

The guide is built on top of the GOV.UK frontend toolkit, the GOV.UK template, and GOV.UK elements.

In particular, GOV.UK elements provides a large number of useful styles. Please get familiar with it!

The Sass files for all the examples in this page can be found on GitHub.

  1. Typography
  2. Data visualisation
  3. Forms
  4. Navigation
  5. Modules (ui elements)
  6. Utilities (dividers, visual breaks etc)
